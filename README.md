# Galymzhan (Ғалымжан) — AI-наставник для предпринимателей

> "Душа Знающего" — назван в честь деда создателя

## Что это

Telegram-бот на базе Groq API (Llama 70B), который помогает предпринимателям:
- Проверить бизнес-идею
- Посчитать unit-экономику
- Построить финансовую модель
- Разработать стратегию

## Быстрый старт

### 1. Получи ключи

**Telegram Bot Token:**
1. Открой [@BotFather](https://t.me/BotFather) в Telegram
2. Отправь `/newbot`
3. Введи имя бота (например: Galymzhan Mentor)
4. Введи username (например: galymzhan_mentor_bot)
5. Скопируй токен

**Groq API Key:**
1. Зайди на [console.groq.com](https://console.groq.com)
2. Зарегистрируйся (бесплатно)
3. Перейди в [API Keys](https://console.groq.com/keys)
4. Создай новый ключ
5. Скопируй его

### 2. Настрой окружение

```bash
# Клонируй или скопируй папку galymzhan-bot

# Создай виртуальное окружение
python -m venv venv

# Активируй (Windows)
venv\Scripts\activate

# Активируй (Linux/Mac)
source venv/bin/activate

# Установи зависимости
pip install -r requirements.txt
```

### 3. Создай .env файл

```bash
# Скопируй пример
cp .env.example .env

# Отредактируй .env и вставь свои ключи
```

### 4. Запусти

```bash
python bot.py
```

## Команды бота

| Команда | Описание |
|---------|----------|
| `/start` | Приветствие |
| `/help` | Справка |
| `/idea` | Анализ бизнес-идеи |
| `/unit` | Unit-экономика |
| `/finance` | Финансовая модель |
| `/strategy` | Стратегия |
| `/clear` | Очистить историю |
| `/stats` | Статистика |

## Технологии

- **LLM**: Groq API (Llama 3.1 70B)
- **Telegram**: aiogram 3.x
- **База данных**: SQLite (aiosqlite)
- **HTTP**: httpx (async)

## Лимиты Groq (бесплатный план)

- 30 запросов в минуту
- 14,400 запросов в день
- Без ограничений на токены

Этого достаточно для тестирования и первых пользователей.
